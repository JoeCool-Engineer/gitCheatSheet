Git Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite Git commands? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code for you to copy.

Check git Version

Check the current version of your local copy of Git by typing the following command:

git --version

Download Repository

Download a remote repository from a url for example gitHup.

git clone <url>

git Status

This command shows the current state of the working directory and the staged changes.

git status

git Add

command adds a change in the working directory to be staged and ready to be committed. If you add a period to the end of the command (for example, git add .), it will add any untracked or modified files in the current directory (the current directory is represented by .) and all subdirectories to be staged.

git add .

git Commit

command captures a snapshot of the currently staged changes.

git commit -m 'initial commit'

git Push

This command pushes up the local repository content to a remote repository. After the files have been added and committed, they must be pushed up to the remote repository on GitHub.

git push origin main
